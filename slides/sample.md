## Marpとは？

- Markdown記法でプレゼンテーションスライドを作成できるツール
- HTMLやPDF、PowerPointファイルに出力可能
- Git管理しやすいテキストベースのスライド

---

## 基本的な使い方

### スライドの区切り

```markdown
---
```

3つのハイフンでスライドを区切ります

---

## コードブロック

```javascript
function hello() {
  console.log("Hello, Marp!");
}
```

シンタックスハイライトも対応しています
